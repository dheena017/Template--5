# 🚀 Frontend Pages (Tool Categories)

**This folder is the core of the application's complexity.** It contains isolated subdirectories for every major category of AI tool.

Categories include:
- `avatar/`: AI Avatar generation and editing.
- `dashboards/`: Multi-tool layouts and user overviews.
- `developers/`: Developer documentation and API keys UI.
- `files/`, `image/`, `pdf/`, `studio/`, `social-media/`, `speech/`, `text/`, `intelligence/`, `marketing/`: Individual pages mapping out tools for that specific category.

Each subfolder contains the UI mapping for those tools. If you are adding a new tool, find its category folder here!\n
