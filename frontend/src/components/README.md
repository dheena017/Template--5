# UI Components

Reusable UI building blocks used to construct the pages.
- `buttons/`: Interactive buttons (Primary, Secondary, Outline).
- `common/`: General widgets like Toggles and Asset Trays.
- `dropdowns/`: Menu overlays.
- `layouts/`: Core structure wrappers (e.g. `ToolLayout.jsx`).
- `sidebars/`: Tool-specific navigation bars.
- `toolSettings/`: The floating right-hand panels where users configure their active tool.\n
