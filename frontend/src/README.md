# React Source (`src/`)

This directory is the heart of the "All AI tools in one website". Because there are so many tools, structure is strictly enforced:

If you want to view a specific feature's UI, go to **`pages/`**.
If you want to edit a button or a sidebar, go to **`components/`**.
If you want to edit how data speaks to the backend, go to **`services/`**.
If you want to tweak shared helper logic (e.g. converting a file locally), go to **`utils/`**.\n
